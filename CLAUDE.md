# Jisr App - Project Rules

## Overview
Jisr (جسر) is an Arabic RTL business management system (ERP-lite dashboard).

## Tech Stack
- **Frontend:** React 18 + Vite
- **Styling:** Tailwind CSS
- **Backend/DB:** Supabase (project ref: `gcvshzutdslmdkwqwteh`)
- **Hosting:** Netlify (auto-deploy from `main` branch)

## Development Rules
- All UI content must be in Arabic with RTL direction
- Database table and column names must be in English
- Use React hooks and functional components only — no class components
- Prefer editing existing files over creating new ones
